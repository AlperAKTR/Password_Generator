** Password Generator **
-------------------------
A simple Python program that creates random, secure passwords based on a chosen length.

Features
----------------------
Generates secure passwords using:

- Lowercase letters
- Uppercase letters
- Numbers
- Symbols
- Safe numeric input validation
- Adjustable password length (6–12)
- Option to generate multiple passwords
- Clean output with colored messages

Usage Example
----------------------

-Welcome to our password generator!

-Please enter your password length (must be 6–12): 10

-Here is your password!: aF9#tL2@Qp

-Wanna generate again? (y/n): y

-Please enter your password length (must be 6–12): 8

-Here is your password!: K3!dA0$p

-Wanna generate again? (y/n): n

-Thank you for using our program :)

How It Works
----------------------
- You choose a password length between 6–12
- The program builds a character pool (letters, numbers, symbols)
- It randomly selects characters from this pool
- The password is assembled and shown to you
- You can generate new passwords as many times as you like
